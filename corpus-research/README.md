# Corpus Research Archive

This directory contains saved results from queries to the Kaleshwar Corpus API.

## Purpose

All corpus searches and generated passages are automatically archived here for:
- Future reference and study
- Building understanding over time
- Creating a personal knowledge base
- Tracking spiritual research and insights

## File Naming

Format: `YYYY-MM-DD_query-topic.md`

Example: `2026-02-06_guru-shishya-relationship.md`

## Contents

Each file includes:
- Query details (endpoint, date, parameters)
- Full passages retrieved
- Source attributions (book codes)
- Key themes extracted

## Usage

These archives can be:
- Read and reviewed during study
- Searched with memory_search
- Referenced in conversations
- Used to track spiritual learning over time
